    1. 操作系统(Operating System , OS):
      1. 是指控制和管理整个计算机系统的硬件和软件资源,并合理组织调度计算机的工作和资源的分配,以提供给用户和其他软件方便的接口和环境,它是计算机系统中最基本的系统软件.(系统资源的管理者),
      2. 因为硬件只听得懂二进制指令,如010101101011010111010101,是不友好的交互接口,在硬件之上安装了操作系统,操作系统对外暴露了友好的交互接口,向上层提供方便易用的服务
    2. 执行一个程序前需要将该数据放到内存中,才能被CPU处理

  • 操作系统如何实现对硬件机器的拓展?
    1. 通常把覆盖了软件的机器称为扩充机器,又称为虚拟机
    2. 操作系统对硬件机器的拓展:将CPU,内存,磁盘,显示器,键盘等已硬件合理地组织起来,让各种硬件能够相互协调配合,实现更多更复杂的功能.

    • 操作系统(Operating System , OS):

    是指控制和管理整个计算机系统的硬件和软件资源,并合理组织调度计算机的工作和资源的分配,以提供给用户和其他软件方便的接口和环境,它是计算机系统中最基本的系统软件.(系统资源的管理者),

    因为硬件只听得懂二进制指令,如010101101011010111010101,是不友好的交互接口,在硬件之上安装了操作系统,操作系统对外暴露了友好的交互接口,向上层提供方便易用的服务

    执行一个程序前需要将该数据放到内存中,才能被CPU处理

    系统资源的管理者:

    • 功能:
      • 处理机管理(对应的进程被处理机(CPU)处理)
      • 存储器管理(需要把程序相关数据放入内存)
      • 文件管理
      • 设备管理(比如qq调用的摄像头功能)
    • 目标:
      • 安全,高效

    操作系统为用户提供了哪些服务?

    • GUI(Graphical User Interface) 图形化用户接口
    • 早期的操作系统:(命令接口)
      • 联机命令接口实例 = 交互式命令接口:
        • cmd -> 比如输入time 就会显示几点几分
      • 脱机命令接口 = 批处理命令接口
      • 搜索c盘中的*.bat文件,
        • 可以理解为一次提出很多请求,操作系统会依次执行这些命令
      • 程序接口:可以在程序中进行系统调用来使用程序接口.普通用户不能直接使用程序接口,只能通过程序代码间接使用
        • 系统调用类似于函数调用,是应用程序请求操作系统服务的唯一方式

    操作系统如何实现对硬件机器的拓展?

    没有任何软件支持的计算机称为裸机,在裸机上安装的操作系统,可以提供资源管理功能和方便用户的服务功能,将裸机改造成功能更强,还是用更方便的机器

    通常把覆盖了软件的机器称为扩充机器,又称为虚拟机

    操作系统对硬件机器的拓展:将CPU,内存,磁盘,显示器,键盘等已硬件合理地组织起来,让各种硬件能够相互协调配合,实现更多更复杂的功能.
